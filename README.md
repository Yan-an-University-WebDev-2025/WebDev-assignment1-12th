# 响应式个人博客系统

## 项目简介

本项目是一个基于前端技术实现的静态博客站点，具备文章展示、分类浏览、评论互动等核心功能。采用响应式设计，可适配桌面端、平板和移动设备，提供良好的跨设备浏览体验。

- CSS3：样式设计，包括 Flexbox 和 Grid 布局
- JavaScript：交互逻辑和数据处理
- 响应式设计：适配不同设备屏幕尺寸

## 功能特点

1. **文章展示**：
   - 首页分页展示文章列表（每页至少6篇文章）
   - 热门文章推荐（按浏览量排序）
   - 文章详情页展示完整内容
   - 支持传统页码导航和"加载更多"按钮双模式
   
2. **分类浏览**：
   - 支持按分类（技术文章、生活随笔、学习笔记）筛选文章
   - 分类导航与标签云
   - 分类页面支持分页展示
   
3. **用户系统**：
   - 用户注册与登录功能（基于 localStorage）
   - 用户信息管理（昵称、邮箱、账号、密码、头像）
   - 登录状态显示与退出功能
   - 评论时显示用户头像
   
4. **用户互动**：
   - 评论展示功能（带用户头像）
   - 评论提交与表单验证
   - 联系表单与消息发送
   
5. **响应式设计**：
   - **移动端（≤768px）**：单列布局，隐藏侧边栏
   - **平板端（769px-1024px）**：双列布局，优化导航
   - **桌面端（>1024px）**：三列布局，显示完整侧边栏
   - 移动端汉堡菜单导航
   
6. **界面美化**：
   - 渐变按钮与动画效果
   - 卡片化设计与阴影效果
   - 输入框焦点高亮
   - Hover 悬浮动画

## 页面说明

1. **首页（index.html）**：
   - 展示文章列表，支持分页加载
   - 页码导航（上一页/下一页）和"加载更多"按钮
   - 侧边栏显示热门文章、分类列表和标签云
   - 响应式三段式布局
   
2. **分类页（category.html）**：
   - 展示各分类简介
   - 提供分类导航入口
   
3. **分类文章列表页（category-list.html）**：
   - 按选定分类展示相关文章
   - 支持分页加载与导航
   
4. **文章详情页（detail.html）**：
   - 展示文章完整内容
   - 评论列表展示（带用户头像）
   - 评论提交功能与表单验证
   
5. **登录注册页（login.html）**：
   - 用户登录功能
   - 用户注册功能（昵称、邮箱、账号、密码、头像）
   - 表单验证与错误提示
   - 美化的宽屏双列布局
   
6. **关于页（about.html）**：
   - 网站介绍与团队信息
   
7. **联系页（contact.html）**：
   - 联系方式展示（邮箱、社交媒体、地址）
   - 消息发送表单
   - 美化的卡片式设计

## 本地运行

1. 克隆或下载项目到本地
2. 使用浏览器直接打开index.html文件
3. 或使用 VSCode 的 Live Server 插件运行（默认端口 5501）

## 主要模块说明

1. **ArticleManager（script.js）**：
   - 文章数据生成与管理（50篇模拟文章）
   - 文章列表分页加载（每页6篇）
   - 页码导航与"加载更多"双模式
   - 热门文章筛选与展示
   
2. **ArticleDetail（detail.js）**：
   - 文章详情加载与展示
   - 评论管理与提交验证
   - 评论头像显示功能
   
3. **CategoryArticleList（category-list.js）**：
   - 分类文章筛选与展示
   - 分类页面分页功能
   
4. **AuthManager（auth.js）**：
   - 用户注册功能（数据存储在 localStorage）
   - 用户登录验证
   - 用户信息管理
   - 登录状态维护
   
5. **GlobalNavigation（global.js）**：
   - 导航栏管理与高亮
   - 移动端汉堡菜单功能
   - 用户登录状态显示
   - 退出登录功能
   
6. **FooterManager（global.js）**：
   - 页脚信息管理与更新

## 开发与扩展建议

1. 集成后端服务（Node.js + MongoDB/MySQL）实现真实数据管理
2. 升级用户认证系统，使用 JWT 或 Session 机制
3. 实现文章搜索功能（全文搜索）
4. 添加文章编辑和发布功能（Markdown 编辑器）
5. 集成数据统计与分析功能（浏览量、点赞、收藏）
6. 添加图片上传功能（头像、文章配图）
7. 实现评论回复与点赞功能
8. 添加主题切换（深色模式/浅色模式）

## 注意事项

- 当前用户数据存储在浏览器 localStorage，清除浏览器数据会导致用户信息丢失
- 文章数据为模拟数据，刷新页面不会保存新发布的内容
- 评论功能暂未持久化，刷新页面后评论会恢复为默认数据
- 建议使用现代浏览器（Chrome、Firefox、Edge）以获得最佳体验
