# 响应式个人博客系统

## 项目简介

本项目是一个基于前端技术实现的静态博客站点，具备文章展示、分类浏览、评论互动等核心功能。采用响应式设计，可适配桌面端、平板和移动设备，提供良好的跨设备浏览体验。

## 技术栈

- HTML5：页面结构与语义化标签
- CSS3：样式设计，包括 Flexbox 和 Grid 布局
- JavaScript：交互逻辑和数据处理
- 响应式设计：适配不同设备屏幕尺寸

## 功能特点

1. **文章展示**：
   - 首页分页展示文章列表
   - 热门文章推荐（按浏览量排序）
   - 文章详情页展示完整内容
2. **分类浏览**：
   - 支持按分类（技术文章、生活随笔、学习笔记）筛选文章
   - 分类导航与标签云
3. **用户互动**：
   - 评论展示功能
   - 评论提交与表单验证
4. **响应式设计**：
   - 适配桌面、平板和移动设备
   - 移动端汉堡菜单导航

## 页面说明

1. **首页（index.html）**：
   - 展示文章列表，支持分页加载
   - 侧边栏显示热门文章、分类列表和标签云
2. **分类页（category.html）**：
   - 展示各分类简介
   - 提供分类导航入口
3. **分类文章列表页（category-list.html）**：
   - 按选定分类展示相关文章
   - 支持分页加载
4. **文章详情页（detail.html）**：
   - 展示文章完整内容
   - 评论列表展示
   - 评论提交功能
5. **关于页（about.html）**：
   - 网站介绍与团队信息
6. **联系页（contact.html）**：
   - 联系方式展示
   - 消息发送表单

## 本地运行

1. 克隆或下载项目到本地
2. 使用浏览器直接打开index.html文件
3. 或使用 VSCode 的 Live Server 插件运行（默认端口 5501）

## 主要模块说明

1. **ArticleManager（script.js）**：
   - 文章数据生成与管理
   - 文章列表分页加载
   - 热门文章筛选与展示
2. **ArticleDetail（detail.js）**：
   - 文章详情加载与展示
   - 评论管理与提交验证
3. **CategoryArticleList（category-list.js）**：
   - 分类文章筛选与展示
4. **GlobalNavigation（global.js）**：
   - 导航栏管理与高亮
   - 移动端菜单功能
5. **FooterManager（global.js）**：
   - 页脚信息管理与更新
